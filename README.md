TaskFlow - Personal Project Manager
 Project Description
TaskFlow is a personal task management application developed using Java and Java Swing. The project is designed to help users organize their daily tasks efficiently through a simple and user-friendly interface.

 Key Features
Task Management: Easily add, edit, and delete tasks.

Graphical User Interface (GUI): Modern design using the Java Swing library.

Smart Organization: Categorize tasks for easy access and management.

 Technologies Used
Language: Java

Interface: Java Swing

IDE: NetBeans IDE

 Developer

How to Get Started?
Ensure JDK is installed on your machine.

Download or clone the source code from this repository.

Open the project using NetBeans and perform a Build & Run.
